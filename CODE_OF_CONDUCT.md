# Code of Conduct

## Our Commitment

This project is committed to a respectful, harassment-free environment for maintainers, collaborators, and users.

## Expected Behavior

- Be respectful and professional.
- Assume good intent and discuss disagreements with facts.
- Focus feedback on code, behavior, and outcomes.
- Respect security and privacy boundaries.

## Unacceptable Behavior

- Harassment, discrimination, threats, or personal attacks.
- Doxxing or sharing private or confidential information.
- Repeated off-topic disruption or bad-faith arguments.
- Any behavior that makes collaboration unsafe or hostile.

## Reporting

Report conduct incidents privately through repository maintainers using a private security report in GitHub Security Advisories.

Do not post sensitive conduct reports publicly.

## Enforcement

Maintainers may take any action needed to protect the project, including:

- Warning
- Content removal
- Temporary restrictions
- Permanent removal from collaboration channels

Enforcement decisions are made at maintainer discretion based on impact and repeated behavior.
